# V0.4 版本建议

## 推荐新功能

- 增加设置面板：集中管理主题、语言、朗读音色、语速、默认模型等偏好。
- 支持会话搜索和置顶，方便快速找回历史聊天。
- 增加会话导出功能，支持导出为 Markdown、TXT 或 JSON。
- 增加停止生成按钮，AI 回答太长或不符合预期时可以立即中断。
- 支持重新生成上一条回答，并保留多个回答版本。
- 增加消息编辑后重新发送，减少重复复制粘贴。
- 支持图片上传或文件提问，为后续多模态能力做准备。

## 推荐体验优化

- 增加首屏欢迎状态，空会话时显示快捷问题和常用入口。
- 增加“正在输入/生成中”的更细腻状态提示。
- 优化移动端侧边栏为抽屉遮罩，点击内容区自动收起。
- 增加代码块一键折叠、自动换行开关和下载代码功能。
- 增加全局 Toast 提示，替代频繁 `alert` 弹窗。
- 增加本地数据备份和恢复，降低清缓存导致历史丢失的风险。

## 推荐工程优化

- 把 API Key 移到后端代理或环境变量，避免前端直接暴露密钥。
- 拆分核心逻辑：会话管理、消息渲染、语音能力、主题语言分别成模块。
- 增加统一状态管理，减少 `localStorage`、DOM 和内存状态不一致。
- 增加基础测试用例：会话切换、消息保存、语言切换、代码复制。
- 增加离线兜底资源，避免 CDN 加载失败影响 Markdown 和代码高亮。
