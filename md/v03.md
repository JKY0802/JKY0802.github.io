# V0.3 更新记录

## 本次重点

- 全面优化 UI 风格，整体更接近 iOS 的简洁、轻盈、毛玻璃质感。
- 增强移动端适配，顶部栏、输入框、侧边栏和弹窗在小屏下更稳定。
- 新增多会话列表渲染，新建、切换、重命名、删除会话更完整。
- 修复新建会话内容串台问题，每个会话独立保存历史。
- 修复发送后马上切换会话导致消息没有保存的问题。
- 发送消息后自动滚动到底部，流式回答过程中也会持续跟随最新内容。
- 侧边栏收起后显示会话短标签，并高亮当前会话。
- 中英文切换同步更新新建会话、主题模式、代码复制等文案。
- 朗读优先选择更自然的系统音色，并微调语速和音高。
- AI 回复 Markdown 排版优化，标题、段落、列表、引用、表格更易读。
- 代码块增加语言标签和独立复制按钮，支持复制代码块内容。
- 新增 V0.3 更新公告，并保留 V0.2 公告注释备份。

## 文件调整

- 新增 `css/v03.css`：集中放置 V0.3 样式增强。
- 新增 `js/v03.js`：集中放置 V0.3 交互和功能增强。
- 更新 `index.html`：接入 V0.3 资源，更新公告内容。

## 已知说明

- 朗读音色取决于浏览器和系统实际安装的语音包。
- 代码高亮依赖 `highlight.js` 和网络 CDN，离线时会保留基础代码块样式。
